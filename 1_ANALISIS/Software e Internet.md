## Software e Internet
Se define software como el conjunto de componentes lógicos, que hacen posible la realización de tareas específicas.

### Tipos de Software
1. Software de Sistema

Está constituido por los programas que se encargan del control y administración de los recursos de cómputo.
Permiten la interacción entre los usuarios y los sistemas de Aplicación (Hardware).
Ejemplos: traductores(compliadores, intérpretes, ensambladores) o Sistemas Operativos.

2. Software de Aplicación

Son los programas que permiten a una computadora realizar tareas específicas de procesamiento de información.
Ejemplos: Editores de texto, hojas de cálculo, antivirus, powerpoint.

3. Software de Programación

Permite a determinados usuarios la comunicación con la máquina o dispositivo para el desarrolo de programas que facilitan el manejo de la información, y permite la toma de decisiones.(Lenguajes de Programación).

Se clasifican en: Alto, Medio, Bajo en relación con su comprensión y su cercanía a la máquina. ( El nivel bajo o  lenguaje máquina, es utilizado para la creación o administración del software de sistema)

El nivel alto para la creación de aplicaciones comerciales o científicas.
El software de programación se presenta en Entornos Integrados de Desarrollo (IDE) tales como: Netbeans, Eclipse, Visual Studio Code, Sublime, Atom, Cursor.
